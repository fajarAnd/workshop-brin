OVERVIEW
=============================================================================

Tujuan: Panduan langkah-demi-langkah untuk melakukan password reset dan account unlock untuk akun Active Directory BRIN

Scope:
- Password reset untuk Windows domain account
- Email password reset (Office 365)
- Account unlock setelah multiple failed login attempts
- VPN account password reset

Prerequisites:
- User harus sudah terverifikasi identitasnya
- User memiliki akses ke email pemulihan atau nomor HP terdaftar
- Untuk password admin, diperlukan approval dari atasan langsung

=============================================================================
A. SELF-SERVICE PASSWORD RESET (USER MANDIRI)
=============================================================================

Metode 1: Portal Self-Service (Recommended)

Langkah 1: Akses Portal Password Reset
- Buka browser dan kunjungi: https://passwordreset.brin.go.id
- Atau dari halaman login Windows, klik "Forgot Password"

Langkah 2: Verifikasi Identitas
- Masukkan username BRIN Anda (format: namauser@brin.go.id)
- Klik "Send Verification Code"
- Pilih metode verifikasi:
  * Email pemulihan (recovery email)
  * SMS ke nomor HP terdaftar
  * Microsoft Authenticator app

Langkah 3: Masukkan Kode Verifikasi
- Cek email atau SMS Anda
- Masukkan 6-digit verification code
- Kode valid selama 10 menit
- Jika kode expire, request kode baru

Langkah 4: Buat Password Baru
- Password minimal 12 karakter
- Harus mengandung:
  * Minimal 1 huruf besar (A-Z)
  * Minimal 1 huruf kecil (a-z)
  * Minimal 1 angka (0-9)
  * Minimal 1 karakter spesial (!@#$%^&*)
- Password tidak boleh mengandung:
  * Nama user atau nama lengkap
  * Kata "password", "brin", "admin"
  * Password yang pernah digunakan sebelumnya (5 password terakhir)

Langkah 5: Konfirmasi dan Simpan
- Masukkan password baru 2 kali untuk konfirmasi
- Klik "Reset Password"
- Tunggu konfirmasi sukses
- Logout dari semua devices dan login ulang dengan password baru

Waktu Estimasi: 5-10 menit

Langkah 3: Temporary Password
- IT Support akan generate temporary password

Langkah 4: First Login dengan Temporary Password
- Login ke Windows atau portal dengan temporary password
- Sistem akan meminta ganti password immediately
- Ikuti kebijakan password complexity (lihat Metode 1 - Langkah 4)

Waktu Estimasi: 15-20 menit

=============================================================================
B. ACCOUNT UNLOCK (LOCKED DUE TO FAILED LOGIN ATTEMPTS)
=============================================================================

Penyebab Account Lock:
- 5 kali salah password dalam 30 menit
- Suspicious login activity
- Policy violation (manual lock by admin)

Metode 1: Automatic Unlock (Tunggu 30 Menit)
- Account akan otomatis unlock setelah 30 menit
- Tidak perlu hubungi IT Support
- Gunakan waktu tunggu untuk pastikan password yang benar
- Jika urgent, gunakan Metode 2

Metode 2: Manual Unlock via IT Support

Langkah 1: Hubungi IT Support
- Telepon: (021) 8012-3456 ext. 123
- WhatsApp: +62 812-3456-7890
- Sebutkan: "Account saya terkunci (locked), mohon unlock"

Langkah 2: Verifikasi Identitas
- Sama seperti prosedur password reset (lihat bagian A - Metode 2 - Langkah 2)

Langkah 3: IT Support Unlock Account
- IT Support akan cek status account di Active Directory
- Unlock account secara manual
- Konfirmasi ke user bahwa account sudah unlocked

Langkah 4: Test Login
- User mencoba login kembali
- Jika masih gagal, mungkin password salah → lakukan password reset

Waktu Estimasi: 5-10 menit

C. PASSWORD RESET UNTUK SISTEM SPESIFIK
=============================================================================

C1. Email Password (Office 365)

Catatan: Password Office 365 sama dengan Active Directory (Single Sign-On)

Jika hanya email yang bermasalah:
1. Buka https://outlook.office.com
2. Klik "Forgot Password"
3. Ikuti prosedur Self-Service Password Reset (bagian A - Metode 1)
4. Setelah reset, logout dan login ulang di:
   - Outlook desktop app
   - Outlook mobile app
   - Semua devices yang tersinkronisasi

---

C2. VPN Password Reset

Langkah 1: Akses VPN Portal
- Buka https://vpn.brin.go.id
- Klik "Forgot VPN Password"

Langkah 2: Verifikasi
- Masukkan VPN username (biasanya sama dengan domain username)
- Masukkan email pemulihan
- Kode verifikasi dikirim ke email

Langkah 3: Reset Password
- Ikuti password complexity policy (sama dengan domain password)
- Password VPN berbeda dari domain password untuk security

Langkah 4: Test Koneksi VPN
- Buka VPN client (Cisco AnyConnect / FortiClient)
- Masukkan username dan password baru
- Test koneksi ke jaringan BRIN

Waktu Estimasi: 10 menit

D. TROUBLESHOOTING - MASALAH UMUM
=============================================================================

Masalah 1: "Password salah" padahal yakin sudah benar
Solusi:
- Pastikan CAPS LOCK tidak aktif
- Pastikan NUM LOCK aktif (jika password ada angka)
- Coba ketik password di Notepad dulu untuk cek visibility
- Pastikan keyboard language setting benar (Indonesia/English)

Masalah 2: Password reset berhasil, tapi tidak bisa login
Solusi:
- Tunggu 5-10 menit untuk replikasi Active Directory
- Logout dari semua devices dulu
- Clear browser cache/cookies
- Restart komputer
- Coba login dari komputer/device lain

Masalah 3: Tidak menerima verification code
Solusi:
- Cek folder Spam/Junk email
- Pastikan email pemulihan masih aktif dan accessible
- Cek nomor HP terdaftar masih aktif
- Request resend code (max 3x)
- Jika masih gagal, hubungi IT Support untuk manual reset

Masalah 4: Password baru tidak memenuhi complexity requirements
Solusi:
- Gunakan password generator: https://passwordgenerator.brin.go.id
- Atau buat password dengan pola: Nama@BRIN2025! (contoh: Budi@BRIN2025!)
- Hindari password umum seperti: Password123!, Admin@2025
- Gunakan password manager untuk simpan (recommended: Bitwarden)

Masalah 5: Account masih locked padahal sudah tunggu 30 menit
Solusi:
- Kemungkinan ada aplikasi/service yang terus mencoba login dengan password lama
- Cek aplikasi yang tersinkronisasi:
  * Outlook di mobile
  * VPN auto-connect
  * Mapped network drives
  * Third-party apps dengan BRIN credentials
- Logout/disconnect semua aplikasi tersebut
- Hubungi IT Support untuk manual unlock

=============================================================================
E. KEBIJAKAN PASSWORD BRIN
=============================================================================

Mandatory Password Policy:
- Minimal panjang: 12 karakter
- Maksimal umur: 90 hari (wajib ganti setiap 3 bulan)
- Notifikasi reminder: 14 hari, 7 hari, 3 hari sebelum expire
- Password history: 5 (tidak boleh reuse 5 password terakhir)
- Account lockout: 5 failed attempts dalam 30 menit

Recommended Best Practices:
- Gunakan passphrase: "SayaSukaBROccoli2025!" lebih baik dari "P@ssw0rd123!"
- Aktifkan Multi-Factor Authentication (MFA) untuk akun email
- Gunakan password manager untuk kelola password
- Jangan share password dengan siapapun, termasuk IT Support
- Jangan tulis password di sticky notes atau simpan di file plaintext
- Gunakan password unik untuk setiap sistem (jangan reuse)

Multi-Factor Authentication (MFA):
- Wajib untuk:
  * Remote VPN access
  * Office 365 email
  * Admin/privileged accounts
- Optional tapi recommended untuk:
  * Windows login
  * Aplikasi internal BRIN

A. Password Complexity Example
- ❌ Bad: password123, admin2025, brin1234
- ❌ Bad: namasaya, Budi1234, Jakarta2025
- ✅ Good: Gunung$em3ru2025!, BukuM3rah@2025
- ✅ Good: K0pi&Teh#Pagi99, 5Apel%Hijau!2025

B. Common Error Messages
- "Your account has been locked out": Account locked, tunggu 30 menit atau hubungi IT
- "Your password has expired": Ganti password dengan yang baru
- "The password does not meet the password policy": Gunakan password lebih kuat
- "Password incorrect": Salah password atau CAPS LOCK aktif
