INTRODUCTION
=============================================================================

Selamat datang di IT Service Catalog BRIN!

Service Catalog ini berisi daftar lengkap layanan IT yang tersedia untuk semua karyawan BRIN. Setiap layanan memiliki deskripsi, cara request, SLA (Service Level Agreement), dan kontak support.

Cara Menggunakan Service Catalog:
1. Temukan layanan yang Anda butuhkan di kategori terkait
2. Baca deskripsi & requirements
3. Submit request via portal atau contact yang tertera
4. Track status request Anda via ServiceNow portal

Service Request Portal: https://serviceportal.brin.go.id
Help Desk: (021) 8012-3456 ext. 123 | itsupport@brin.go.id

=============================================================================

SERVICE: New User Account Creation
-----------------------------------
Deskripsi:
Pembuatan akun Active Directory baru untuk karyawan baru, termasuk email, network access, dan aplikasi standar.

What's Included:
- Active Directory account
- Email (Office 365)
- Network access (WiFi & VPN)
- Standard applications (Office, Teams, OneDrive)
- Access to shared drives (department folder)

Prerequisites:
- HR onboarding completed
- Employee ID assigned
- Manager approval (automatic dari HR system)

How to Request:
- Automatic: HR system triggers account creation saat onboarding
- Manual: Submit request via https://serviceportal.brin.go.id/new-account

SLA:
- Provisioning: 1 business day dari HR approval
- Priority: Standard

Cost: No charge (included in IT budget)

CATEGORY 4: NETWORK & CONNECTIVITY

SERVICE: WiFi Access
--------------------
Deskripsi:
Access ke BRIN corporate WiFi network.

SSID: BRIN-Corporate

Access:
- Automatic untuk devices enrolled di MDM
- Manual: Connect → Login dengan AD credentials

Requirements:
- Active BRIN account
- Device meets security requirements (antivirus, OS updated)

Troubleshooting:
- Cannot connect: Contact IT Support
- Slow performance: Check signal strength, try different AP

Guest WiFi:
- SSID: BRIN-Guest
- Access: Request guest account dari IT Support (for external visitors)
- Duration: Max 8 hours per session, max 7 days validity

SLA: Immediate (self-service), IT Support 30 minutes

Cost: No charge

---

SERVICE: VPN Access
-------------------
Deskripsi:
Remote access ke BRIN network dari luar kantor.

Included:
- VPN client (GlobalProtect)
- Access to internal resources (file shares, applications, intranet)
- Secure encrypted connection

Prerequisites:
- Active BRIN account
- MFA enabled (mandatory)
- VPN access approval (automatic untuk employees, by request for contractors)

Setup:
1. Download VPN client: https://vpn.brin.go.id/download
2. Install & configure (auto-config with BRIN email)
3. Connect: Enter credentials + MFA

Usage:
- Capacity: 1000 concurrent users
- Session timeout: 12 hours (reconnect required)
- Allowed: All employees working remotely

Troubleshooting:
- Connection fails: Check internet, check credentials, contact IT Support
- Slow performance: Check bandwidth, disconnect when not needed

SLA: Setup 1 hour, Support 2 hours

Cost: No charge

SERVICE: Equipment Repair
-------------------------
Deskripsi:
Repair untuk hardware yang rusak atau malfunction.

Covered:
- Laptop / desktop repair
- Printer repair
- Monitor, keyboard, mouse replacement
- Network equipment

Process:
1. Submit ticket via portal atau call IT Support
2. Diagnosis (onsite atau bring to IT room)
3. Warranty check:
   - Under warranty: Free repair via vendor
   - Out of warranty: Repair cost estimate → approval → repair
4. Repair atau replacement
5. Return to user

SLA:
- Diagnosis: 4 hours
- Repair: 3-5 business days (under warranty), 1-2 weeks (out of warranty)

Cost:
- Under warranty: No charge
- Out of warranty: Varies (Rp 500k - Rp 5 juta depending on repair)

Alternative:
- If repair cost >50% of new device → recommend replacement

---

SERVICE: Data Backup & Recovery
-------------------------------
Deskripsi:
Backup data dan recovery untuk lost/deleted files.

Automatic Backups:
- OneDrive for Business: Real-time sync, 30 days version history
- Email: Retention policy 7 years
- Shared drives: Daily backup, 30 days retention
- Servers: Daily backup, 90 days retention

Data Recovery Request:
- Portal: https://serviceportal.brin.go.id/data-recovery
- Include: File name, location, approximate deletion date

SLA:
- Recent files (<7 days): 4 hours
- Older files (7-30 days): 1 business day
- Archive restore (>30 days): 3 business days

Cost:
- Standard recovery: No charge
- Large data recovery (>100GB): Rp 500k per 100GB

Prevention:
- Use OneDrive for important files (auto-backup)
- Don't store critical data only on local drive (laptop can fail/be stolen)
