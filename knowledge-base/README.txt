üé´ "User cannot send emails (mailbox full)"
   ‚Üí READ: incident-problem-records.txt
   ‚Üí REFERENCE: INC-2024-0234 - Cannot Send Emails (Mailbox Full)
   ‚Üí SOLUTION: Enable online archive, auto-archive policy, cleanup

üé´ "VPN connection fails"
   ‚Üí READ: incident-problem-records.txt
   ‚Üí REFERENCE: INC-2024-0789 - VPN Connection Fails (Certificate Expiry)
   ‚Üí OR: INC-2024-0654 - VPN Slow Performance (MTU Mismatch)
   ‚Üí CHECK: Certificate validity, MTU settings, VPN capacity

üé´ "WiFi keeps disconnecting"
   ‚Üí READ: incident-problem-records.txt
   ‚Üí REFERENCE: INC-2024-0445 - WiFi Connection Drops (Channel Interference)
   ‚Üí SOLUTION: Wireless site survey, reconfigure channels, adjust power

üé´ "Phishing email reported"
   ‚Üí READ: incident-problem-records.txt
   ‚Üí REFERENCE: INC-2024-0888 - Phishing Attack (Credential Harvesting)
   ‚Üí ACTION: Notify Security Team, force password reset, check for compromise

üé´ "Server disk failed"
   ‚Üí READ: incident-problem-records.txt
   ‚Üí REFERENCE: INC-2024-0345 - Server Disk Failure (RAID Degraded)
   ‚Üí ACTION: Log vendor support case, hot-swap disk, monitor rebuild

TRAINING & CERTIFICATION
=============================================================================

New IT Support Staff Onboarding:
- Day 1: Read SOP, Service Catalog, User Manual Office 365
- Day 2-3: Shadow senior support staff
- Week 2: Read Incident Records, practice troubleshooting
- Week 3-4: Handle tickets with supervision
- Month 2: Full independence, specialize in areas

- All IT Support: Mandatory annual training on knowledge base usage
- How to search effectively
- How to update incident records
- When to create new documentation

Certification:
- Knowledge Base Contributor: Complete training + submit 5 quality incident records
- Benefits: Recognition, eligibility for senior support roles
